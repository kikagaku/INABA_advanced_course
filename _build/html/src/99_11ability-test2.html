【補足】 実力テスト - 機械学習セクション

ポルトガルのある銀行のダイレクトマーケティングキャンペーンに関連したデータセットを使用します。

背景として顧客にダイレクトマーケティング（電話）をかけると1件辺り500円ほどのコストが発生するため、なるべく確度の高い顧客を見逃さないように当たりをつけてダイレクトマーケティング（電話）したいと考えています。キャンペーンの結果、どの顧客が定期預金を申し込むかを予測する（success, fail）モデルを作成してください。

データセット

訓練・検証用のデータセットは bank_train.csv を使用してください。
テスト用のデータセットは bank_test.csv を使用してください。

結果を提出する際は、推論用データセット bank_pred.csvを使用し、推論した結果を success, fail の文字列に変換し csv ファイルで提出してください。

ゴール

推論用のデータセットに対して、Recall が最も高いモデルを構築してください。また、推論結果を success,fail の文字列に変換し、csv ファイルにて提出してください。 文字列の変換に関しては、以下のコードを参考にしてください。

# 推論結果を、 success, fail に書き換える
y_list = []
for y in y_pred:
  if y == '0':
    f = y.replace("0", 'fail')
    y_list.append(f)
  else:
    s = y.replace("1", 'success')
    y_list.append(s)

csv ファイルの提出に関しては、以下のコードを参考にしてください。

# Mac の方
pd.Series(y_list, name='pred').to_csv('submission.csv', index=None),
# Windows の方
pd.Series(y_list, name='pred').to_csv('submission.csv', index=None, line_terminator='\n')
