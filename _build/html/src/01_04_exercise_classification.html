問題設定：Bank Marketing

ここまで実践してきた内容を、実データに対して実践してみましょう。データは、ポルトガルの銀行機関のダイレクトマーケティングキャンペーン（電話）に関連しています。

クライアントが定期預金を登録するかどうかを予測する分類モデルを構築する問題となります。

詳細

データセット
学習と検証用のデータセットはこちらからbank_train.csvを使用してください。
ゴール
テストデータに対して、Accuracy がもっとも高いモデルの構築

POINT

  • 前処理

  • 手法の選択

  • モデルの最適化

  • 評価の考察

入力変数の説明

列名

説明

age

年齢

job

職業

marital

既婚/未婚

education

学歴

default

債務不履行の有無

balance

預金の年間平均残高（単位：ユーロ）

housing

住宅ローンの有無

loan

個人ローンの有無

pdays

前回のキャンペーンで通話してからの経過日数（単位：日）

previous

本キャンペーン以前に通話した回数（単位：回）

poutcome

前回キャンペーンの成約状況

result

本キャンペーンの結果（１⇒成約、０⇒不成約）

注意

学習モデルが構築できたら、テストデータに対する精度で評価します。

テストデータは、こちらからダウンロードしてください。
